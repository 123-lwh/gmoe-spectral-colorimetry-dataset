**"Spectral Colorimetry Based on GMoE for Temperature Measurement in Explosion Fields"**.

## 1. YOLO Detection Training

The YOLO detection training code is provided in:

preprocessing_reference/yolo_train.py

The YOLO label files are provided as:

preprocessing_reference/yolo_dataset/labels.zip

The trained YOLO weight is provided as:

preprocessing_reference/yolo_weight/best.pt

The original BMP image dataset is provided through the download link above. If users want to retrain the YOLO detector, please unzip labels.zip and modify the dataset paths in yolo_train.py according to the local image and label locations.

## 2. YOLO Detection Inference and Position Information

The YOLO detection script is provided as:

preprocessing_reference/yolo_detect

This script loads the trained YOLO weight and detects the circular channel regions from the BMP images.

Before running this script, please modify the local path variables in the code, including the model path, image folder, and output folder.

The detected position information used in this work is also provided as:

preprocessing_reference/data/detection_data.csv

This CSV file contains the detected channel position information, including image filename, temperature, channel position, center coordinates, and bounding-box coordinates.

## 3. Dataset Split Reconstruction

The dataset split file is provided as:

preprocessing_reference/data/split_info.csv

It contains two columns:

name,status

where name is the BMP image filename, and status indicates whether the image belongs to the train, val, or test subset.

The script for reconstructing the train/validation/test image folders is provided as:

preprocessing_reference/prepare_split_folders.py

Before running this script, please modify the following path variables according to the local directory:

ORIGINAL_IMAGE_DIR = Path(r"path/to/all_original_images")
SPLIT_CSV_PATH = Path(r"path/to/preprocessing_reference/data/split_info.csv")
OUTPUT_SPLIT_DIR = Path(r"path/to/output_split_dataset")

Then run:

cd preprocessing_reference
python prepare_split_folders.py

After running the script, the original BMP images will be copied into the following structure according to split_info.csv:

output_split_dataset/
├── train/
├── val/
└── test/
## 4. Polynomial Fitting

The polynomial fitting code is provided as:

preprocessing_reference/poly.py

Before running poly.py, please modify the dataset path and detection CSV path according to the local directory.

In this work, the polynomial fitting procedure needs to be run six times for the six wavelength/channel pairs. For each run, the wavelength parameters and the corresponding channel position information in the code should be modified accordingly.

The detected position information required by the polynomial fitting code is provided in:

preprocessing_reference/data/detection_data.csv

The reconstructed train, val, and test image folders generated by prepare_split_folders.py can be used as the image input for poly.py.

## 5. Transformer-MoE Experiment

The Transformer-MoE model code is provided in:

transformer_moe/

The grayscale feature table used by the Transformer-MoE model is provided as:

transformer_moe/data/moe_test_1.csv

This CSV file contains the data required by the Transformer-MoE model. The dataset split information is included in the table.

The trained weights of ten repeated runs are provided in:

transformer_moe/weights/

Before running the Transformer-MoE code, please modify the path variables in train.py and test.py according to the local directory, especially the CSV file path and the output directory.

To evaluate the provided trained weights, run:

cd transformer_moe
python test.py

To retrain the Transformer-MoE model using the provided grayscale feature table, run:

cd transformer_moe
python train.py
## Notes

The main Transformer-MoE experiment can be run using the provided grayscale feature table, model code, and trained weights after modifying the local paths in the scripts.

The YOLO detection code, YOLO labels, YOLO weight, detected position CSV, dataset split CSV, and polynomial fitting code are provided to document the preprocessing workflow used before the Transformer-MoE stage.
